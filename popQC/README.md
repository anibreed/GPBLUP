# popQC - Population Quality Control Program

## Purpose
Population Quality Control (popQC) program performs comprehensive quality control analysis on genomic SNP data generated by ReadFR program.

### Input
- GENO file (binary genotype data from ReadFR)
- PED file (pedigree information)
- Parameter file (QC thresholds and settings)

### Output
- QC statistics and metrics
- Filtered genotype data
- Sample and SNP quality reports

## Quick Start

### 1. Compile
```bash
cd /home/dhlee/GPBLUP/popQC
make
```

### 2. Run
```bash
./popQC parameter_file.txt
```

### 3. Parameter File Format
```
GENO_FILE = /path/to/ReadFR_output.geno
PED_FILE = /path/to/pedigree.txt
MAP_FILE = /path/to/map.txt

# QC Thresholds
MIN_CALL_RATE_ANIMAL = 0.95
MIN_CALL_RATE_SNP = 0.95
MAX_MAF_RANGE = 0.01 0.50
MIN_HWE_PVALUE = 1e-6
```

## Features

- [ ] Call rate calculation (per animal and per SNP)
- [ ] Allele frequency analysis
- [ ] Hardy-Weinberg equilibrium test
- [ ] Missing data patterns
- [ ] Sample filtering
- [ ] SNP filtering
